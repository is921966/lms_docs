### Метрики и размеры

```
Общие:
- Высота ячейки: 76pt
- Размер аватара: 52x52pt
- Отступ от края: 12pt
- Вертикальный отступ: 8pt

Шрифты:
- Заголовок: SF Pro Display Semibold 16pt
- Превью: SF Pro Text Regular 15pt
- Время: SF Pro Text Regular 14pt
- Badge: SF Pro Text Medium 14pt

Badge:
- Минимальная ширина: 22pt
- Высота: 22pt
- Радиус скругления: 11pt
- Отступ текста: 6-8pt

Анимации:
- Появление: 0.3s ease-out
- Свайп: 0.2s spring
- Обновление: 0.4s ease-in-out
```

## 📱 Специфика для LMS

### Приоритизация новостей
1. **Критические** - красный badge, всегда сверху
2. **Важные** - оранжевый индикатор
3. **Обычные** - стандартное отображение
4. **Архивные** - серый текст

### Категории с иконками
- 🚨 Срочные объявления
- 📚 Новые курсы
- 🎯 Цели и KPI
- 🏆 Достижения команды
- 📅 Корпоративные события
- 💼 Кадровые изменения
- 📊 Отчеты и аналитика
